# 로또 애플리케이션

1. 기본 기능

   - 사용자는 랜덤한 로또 번호를 생성할 수 있어야 한다.
   - 1회에 6개의 번호(1~45 사이의 숫자)로 이루어진 로또 번호를 생성한다.
   - 한 번에 여러 개의 로또 번호를 생성할 수 있어야 한다.
   - 중복된 숫자는 포함되지 않아야 한다.

2. 당첨 확인 기능
   - 당첨 번호(관리자가 입력 또는 API를 통해 제공)를 입력하면, 사용자가 가진 번호들과 비교하여 당첨 여부를 확인한다.
   - 1등(6개 일치), 2등(5개 + 보너스 번호 일치), 3등(5개 일치), 4등(4개 일치), 5등(3개 일치) 등 당첨 등급을 표시한다.
   - 당첨 여부를 확인할 수 있도록 UI에 명확한 표시가 있어야 한다.
